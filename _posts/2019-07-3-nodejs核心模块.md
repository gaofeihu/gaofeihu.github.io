# npm包管理

## npm常用命令




  - npm -v 

       - 查看npm的版本
- npm version 

  - 查看所有模块的版本
- npm search 包名 

     - 搜索包
- npm install / i 包名 

     - 安装包
- npm remove / r 包名 

     - 删除包
- npm install 包名 --save 

     - 安装包并添加到依赖中 *****
- npm install 

     - 下载当前项目所依赖的包
- npm install 包名 -g 

     - 全局安装包（全局安装的包一般都是一些工具）
- npm init 

     - 生成package.json文件 ，这个文件是整个项目的描述文件，通过这个文件可以知道项目依赖关系、版本、作者等其他信息。
- npm config set registry 镜像地址



nvm 是一个node 版本管理的工具，可以随时切换node版本。
